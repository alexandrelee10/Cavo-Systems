## Digital Portfolio Website

A modern, responsive website built with **React** and **Tailwind CSS**, designed to showcase digital projects, services, and clean, interactive UI components. This project focuses on clarity, performance, and thoughtful front-end architecture.

## Features

- Responsive layout for all screen sizes
- Reusable component-based architecture
- Interactive UI elements with hover and cursor-based effects
- Clean, modern design using Tailwind CSS
- Organized sections for services, work, and team
- Dark mode support

## Built With

- **React** – Component-based UI development
- **Tailwind CSS** – Utility-first styling
- **JavaScript / TypeScript**
- **Vite** – Fast development environment


## Notes:

Run npm i first before running dev
